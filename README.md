pour utiliser notre programme il faut:
-lancer un terminal git sur ca machine (chercher dans votre barre de recherche "git CMD))
-placer vous dans le fishier qui vous interesse 
-cloner le projet avec la commande "git clone https://github.com/bionucleaire/travail_cpp.git" 
maintenant le projet et sur votre machine et pret a tournée en locale
-écriver sur votre terminal la commande "make" si un message d'erreur apparait et que vous etez sur windows il faudra alors ecrir la cmande suivant "g++ -std=c++23 -Iinclude -o main src/*.cpp src/shapes/*.cpp main.cpp -lgdi32 -luser32"
-si tout marche alors vous verrez un nouveau fishier apparaitre qui s'appele "main.exe" . executer ce fishier.
